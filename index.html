🔑 Cara Dapatkan API Key

🥇 Gemini (Google) — PERCUMA

1. Buka aistudio.google.com/apikey
2. Login guna Google account
3. Klik "Create API Key"
4. Copy key & paste kat input atas

🥈 DeepSeek — Murah (RM2=1M token)

1. Buka platform.deepseek.com/api_keys
2. Register & top-up (min RM10)
3. Klik "Create API Key"
4. Copy key & paste kat input atas

💡 Nota: App ni auto-detect provider dari format key.
